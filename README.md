# Hanzi (kanji) phonetics research

### 1. Create python virtual environments (optional)
* #### Create
````
python3 -m venv venv
````
* #### Activate
````
source venv/bin/activate
````
* #### Deactivate
````
deactivate
````
### 2. Install requirements
````
pip install -r requirements.txt
````

### 3. Usage example
````
python3 phonetics.py 感
````
