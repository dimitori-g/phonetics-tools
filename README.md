# Hanzi (kanji) phonetics research

### Use python virtual environments (optional)
#### Create
python3 -m venv venv
#### Activate
source venv/bin/activate
#### Deactivate
deactivate

### Installation
pip install -r requirements.txt

### Usage example

python3 phonetics.py 感
